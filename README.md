# Baseball Road Trip Planner

A shared itinerary app for our baseball road trip. Open the link below to view and edit the trip schedule — all changes sync in real time for everyone.

## 🔗 Live App
[Open the planner](https://yourusername.github.io/baseball-road-trip)

## Features
- Add trip days with city and notes
- Log events: games, flights, drives, hotels, food, and sightseeing
- Hotel directory with addresses, confirmation numbers, and phone numbers
- Real-time sync — everyone sees changes instantly

## Trip Details
Add your trip dates and destinations here so everyone knows the plan at a glance.
